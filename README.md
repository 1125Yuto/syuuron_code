# syuuron_code: Code for Master's Thesis

This repository contains the implementation used in my master's thesis on fungal image classification.

## Contents

- `Stratified_k-fold.py`: Stratified K-fold split  
- `cross_validation_CNN_multi_csv_sche_earst.py`: Model training & evaluation  
- `visualize_tsne_umap_from_best.py`: t-SNE / UMAP visualization  
- `premutation_test.py`: Permutation test  
- Other supporting Python scripts for preprocessing and analysis

## Requirements

The code is written in Python and depends on the following (examples):
